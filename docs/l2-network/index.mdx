# HAQQ L2 Network

HAQQ Network L2 is a Layer 2 solution built on the [Optimism](https://github.com/ethereum-optimism/optimism) stack. This solution provides high performance and low fees for HAQQ Network users.

## Testethiq Testnet

Testethiq is the HAQQ L2 testnet that uses Sepolia as the base L1 network.

### Network Parameters

- **Chain ID**: 853211
- **RPC URL**:  https://rpc.testnet.ethiq.network/
- **Explorer**: [https://explorer.testnet.ethiq.network](https://explorer.testnet.ethiq.network/)
- **Block Time**: 2 seconds
- **Gas Fees**: < 0.1 Gwei
- **Bridge**: [https://shell.haqq.network/bridge](https://shell.haqq.network/bridge)
- **Gas limit**: 60 000 000
- **TPS max**: 1428 tx/s 

### Component Versions

HAQQ L2 Testethiq uses the following versions of Optimism stack components:

- **op-node**: [v1.16.1](https://github.com/ethereum-optimism/optimism/releases/tag/op-node%2Fv1.16.1)
- **op-geth**: [v1.101603.4](https://github.com/ethereum-optimism/op-geth/releases/tag/v1.101603.4)
- **op-batcher**: [v1.16.2](https://github.com/ethereum-optimism/optimism/releases/tag/op-batcher%2Fv1.16.2)
- **op-proposer**: [v1.10.0](https://github.com/ethereum-optimism/optimism/releases/tag/op-proposer%2Fv1.10.0)
- **op-contracts**: [v4.0.0](https://github.com/ethereum-optimism/optimism/releases/tag/op-contracts%2Fv4.0.0)
